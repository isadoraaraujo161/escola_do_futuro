1. O que é o Node.js?
O Node.js é um ambiente de execução do código JavaScript do lado servidor (server side), que na prática se reflete na possibilidade de criar aplicações standalone (autossuficientes) em uma máquina servidora, sem a necessidade do navegador.

2. O Node.js usa qual linguagem de programação?
O Node.js utiliza a linguagem de programação JavaScript
3. O Node.js roda no navegador ou fora dele?
O Node.js roda fora do navegador (no lado do servidor ou diretamente na máquina do usuário).
4. Para que serve o Node.js?
O Node.js serve principalmente para construir aplicações de rede rápidas, escaláveis e em tempo real, permitindo executar JavaScript no lado do servidor (back-end).
5. Qual comando é usado para executar um arquivo Node.js no terminal?
nome_do_arquivo.js

6. O que é o npm?
O npm (Node Package Manager) é o gerenciador de pacotes padrão para Node.js e o maior repositório de software de código aberto do mundo. Ele facilita a instalação, compartilhamento e gerenciamento de dependências, bibliotecas e ferramentas JavaScript, automatizando o controle de versões de projetos.
7. O que significa a palavra módulo no Node.js?
No Node.js, um módulo é um arquivo JavaScript independente (ou um conjunto de arquivos/pastas) que encapsula um pedaço específico de funcionalidade, tornando-o reutilizável e organizado.
8. O Node.js é mais usado no front-end ou no back-end?
O Node.js é predominantemente usado no back-end
9. O que é o arquivo package.json?
O arquivo package.json é um componente fundamental em projetos Node.js e desenvolvimento JavaScript (front-end e back-end), funcionando como o manifesto ou identidade do projeto
10. O Node.js é gratuito ou pago?
O Node.js é totalmente gratuito, podendo ser acessado até mesmo em um navegador.
